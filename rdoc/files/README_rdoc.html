wizardly-examples

Examples for jeffp-wizardly gem

github.com/jeffp/wizardly

Setup

clone the directory 

        git clone git://github.com/jeffp/wizardly-examples.git

change to the directory and install the gems 

        rake gems:install

start the server 

        ./script/server
